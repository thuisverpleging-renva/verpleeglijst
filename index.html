Anthropic API-sleutel — nodig voor foto-import

Vereist voor het automatisch uitlezen van patiëntgegevens uit een screenshot. Haal je sleutel op via console.anthropic.com → API Keys.

🔒 De sleutel wordt alleen lokaal op dit toestel bewaard en rechtstreeks naar Anthropic gestuurd — nooit naar een andere server.
Data exporteren & importeren

Exporteer je patiëntenlijst als JSON-bestand om een backup te maken of over te zetten naar een ander apparaat (bv. je iPhone).

App installeren op iPhone
  1. Open deze app in Safari op je iPhone
  2. Tik onderaan op het Deel-icoon (vierkantje met pijl omhoog)
  3. Scroll naar beneden en tik op "Zet op beginscherm"
  4. Tik op "Voeg toe" rechtsboven
✓ De app werkt daarna ook offline — je data blijft bewaard op je toestel.
